2. Huffman Coding (Frequency-Based)

Assigns variable-length codes based on character frequency. **The most frequent character will always receive the shortest possible code, which can be as short as 1 bit.**

3. LZ77 (Sliding Window)

Uses a **Search Buffer** (4 characters) and a **Lookahead Buffer** (3 characters) to find repeated sequences. Output is a triplet: (Offset, Length, Next Symbol).

4. LZ78 (Dictionary Based)

Builds an incremental dictionary of encountered phrases. Output is a pair: (Dictionary Index, Next Symbol).
